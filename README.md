# 🌍 AI Language Translation Tool

## 📌 Overview

The AI Language Translation Tool is a simple Natural Language Processing (NLP) application that translates text from one language to another using Google's pre-trained translation model through the `deep-translator` library.

This project allows users to:
- Enter text for translation.
- Select source and target languages.
- Translate text instantly.
- Perform multiple translations in one session.

---

## 🚀 Features

- Translate text between multiple languages
- Supports English, Hindi, French, Spanish, German, Chinese, Japanese, Korean, Punjabi, Italian, Russian, and Arabic
- Easy-to-use interface
- Fast and accurate translation
- Multiple translations in one execution

---

## 🛠️ Technologies Used
